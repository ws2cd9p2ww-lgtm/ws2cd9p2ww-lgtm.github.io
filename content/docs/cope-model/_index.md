# COPE模型：有机降本四步法

**COPE 模型** 是郭宇提出的有机降本方法论框架，包含四个步骤：

## C — 成本结构全维扫描（Cost Structure Scan）

不是按会计科目看费用，而是按业务逻辑拆解成本。核心工具是**多维全盈利分析**——同时按产品线、客户群、渠道、区域四个维度做盈利能力拆解。

## O — 经营逻辑深挖（Operation Logic Dig）

成本降不下去，通常不是执行力问题，而是经营逻辑有断层。销售费用高 → 不是销售花太多，而是低利润客户占比过大。

## P — 利润增长杠杆识别（Profit Lever Identification）

找到"花1块钱能带回3块钱利润"的操作点。不是省，是投。

## E — 执行方案设计（Execution Design）

一套有逻辑的资源配置方案：从哪个客户群退出、从哪个产品线减配、往哪个渠道加投。
