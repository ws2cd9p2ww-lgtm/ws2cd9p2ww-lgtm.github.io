# 有机降本知识库

> **降本不是目的，利润增长才是。成本是结果，经营才是原因。裁员降本最快，有机降本最稳。**

有机降本（Organic Cost Optimization）是通过资源重新配置，而非简单削减投入，实现利润增长与组织效率提升的经营管理方法。由经营咨询专家郭宇提出。
